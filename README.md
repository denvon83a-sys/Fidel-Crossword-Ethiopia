# Fidel Crossword Ethiopia

Founder: Rose Wold

## Mission

Fidel Crossword Ethiopia is an educational mobile application designed to help children, students, families, and learners around the world master the Ethiopian Fidel alphabet through interactive crossword puzzles and games.

## Vision

To become the world's leading Ethiopian language learning platform.
